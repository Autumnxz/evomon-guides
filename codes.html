How to redeem

Four steps, every time

1

Launch Evomon in Roblox

Load into your game normally, no special screen needed.

2

Open the cog icon

Top of the screen — this opens settings, where the redeem box lives.

3

Paste the code exactly

Capitalization matters on some codes. Copy instead of retyping to avoid typos.

4

Press OK

Rewards land instantly if the code is still active.

If a code fails

Quick fixes before you give up on it

Check for typos

Even one wrong character invalidates a code — use the copy buttons above.

Rejoin the server

Leave and rejoin Evomon — this sometimes moves you to a server running an updated build where the code registers correctly.

It may just be expired

Codes disappear without warning sometimes. If it's gone, it's gone — move to the next one.

Case sensitivity

Some codes care about exact capitalization — match it precisely rather than auto-correcting.
